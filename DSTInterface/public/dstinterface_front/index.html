Instruções:

O cluster é um "save" do jogo, ou seja, cada cluster tem seu próprio mapa.

Para iniciar um cluster insira o nome dele na caixa de texto e aperte o botão "Start/Restart".

O botão "Close Cluster" finaliza o último cluster iniciado.

Só será possível iniciar um cluster por vez.
